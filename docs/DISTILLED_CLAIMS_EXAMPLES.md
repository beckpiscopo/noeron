## Prompt Engineering Insights

### What Makes the Prompt Effective

1. **Concrete Examples** (8 before/after pairs)
   - Shows both good and bad examples
   - Demonstrates the style through repetition

2. **Explicit Rules**
   - NO meta-commentary
   - NO hedging (unless crucial)
   - START with surprise
   - Use ACTIVE voice

3. **Length Enforcement**
   - max_output_tokens=50 (hard limit)
   - Retry with stronger emphasis if >15 words
   - Temperature 0.7 for creativity within constraints

4. **Context Richness**
   - Transcript quote (what was said)
   - Paper title (academic framing)
   - Abstract (broader context)
   - Excerpt (specific finding)

### Common Failure Modes & Fixes

**Too Academic:**
```
âŒ "Bioelectric gradients facilitate morphological determination"
â†’ Increase temperature to 0.8 for more conversational tone
```

**Too Vague:**
```
âŒ "Cells can make decisions"
â†’ Add more paper excerpt context (increase from 800 to 1200 chars)
```

**Too Long:**
```
âŒ "Researchers discovered that planarian worms retain memories even after regeneration" (11 words but verbose)
â†’ Reduce max_output_tokens from 50 to 30
â†’ Add retry logic with penalty
```

**Missing the Punch:**
```
âŒ "Planarians can regenerate their brains"
âœ“ "Planarian worms retain memories after brain removal"
â†’ Add more examples emphasizing surprising/counterintuitive
```
